# AI Career Navigator - Data Analyst Project

## Tools Used
Python, SQL, Power BI

## About Project
I worked on this project to understand job market trends like salary, hiring companies, cities, and work modes.

## What I did
- Cleaned and analyzed dataset using Python (Pandas)
- Used SQL to run basic queries on job data
- Built interactive dashboard using Power BI
- Added filters and charts for better insights

## Key Insights
- Found top hiring cities and companies
- Compared salaries based on experience
- Analyzed remote vs onsite job trends
